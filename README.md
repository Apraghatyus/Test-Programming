# Test-Programming

Moure's roadmap about test to be a best programmer

## Features
- Learn syntax, variables, and data types.
- Examples in TypeScript and Python.
- Beginner-friendly roadmap to improve programming skills.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Test-Programming.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Test-Programming
   ```

## Usage
- Open the `00-Sintax-Variables-Datatype` folder to explore examples.
- Run the Python script:
   ```bash
   python main.py
   ```
- Run the TypeScript file:
   ```bash
   npx ts-node index.ts
   ```
